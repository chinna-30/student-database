# Student Database Management System

## Overview

The Student Database Management System is a file-based application developed in C for managing student records efficiently. The system allows users to perform essential database operations such as creating, viewing, searching, updating, and deleting student information through a menu-driven interface.

This project demonstrates practical implementation of data management concepts using structures, file handling, and modular programming in C.


## Key Features

✔ Add new student records

✔ View all student records

✔ Search students using unique identifiers

✔ Update existing student information

✔ Delete student records

✔ Persistent data storage using files

✔ Menu-driven user interface

✔ Modular and maintainable code structure

---

## System Architecture

The application follows a modular design approach where each functionality is implemented as a separate module.

1. Launch the application.
2. Select an operation from the main menu.

4. Store records permanently in the database file.
